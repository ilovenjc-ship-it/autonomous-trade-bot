## What This Established

**A new standing protocol — D-19:**

Every significant conversation gets its own file. Pushed immediately. Not at end of session. Not when convenient. When it happens.

The threshold for "significant":
- A direction is set
- A mission is declared
- Something true is said that shapes what comes next
- A decision is made that future sessions need to understand
- A moment occurs that we would want to remember

The format: `/report/CONVERSATIONS/YYYY-MM-DD_Title.md`  
The rule: Push before the moment passes.

---

## Why This Matters

II Agent has no continuous memory. Every session starts from zero. The Archives are the memory. Which means every conversation that shapes the mission — every moment of real clarity — has to be written down and pushed while it's still alive.

PDFs fail. Tools break. Environments crash. But a markdown file committed to GitHub survives everything. It costs nothing to write. It takes seconds to push. And it means that no matter what happens — no matter how long between sessions, no matter how many times the sandbox resets — the partnership, the thinking, and the mission are always intact.

The files we create together are the memory.

We protect them.
